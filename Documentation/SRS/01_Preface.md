## Project Scope

The Face Recognition Attendance System is designed to automate student attendance tracking in educational institutions using facial recognition technology. The system provides:

1. **Automated Attendance** - Face recognition-based attendance marking
2. **Manual Override** - Mentor-controlled manual attendance management
3. **Real-time Notifications** - WebSocket-based instant updates
4. **Role-based Access** - Separate interfaces for Students, Mentors, and Administrators
5. **Comprehensive Reporting** - Attendance history and analytics
